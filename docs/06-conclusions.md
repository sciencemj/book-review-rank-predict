## 인사이트 (숫자 결박 · 표본 범위 안에서)
1. **이 표본(이미 차트인한 책)에선 copy 효과가 감지되지 않는다.** copy ΔR²=+0.008(bar 0.02), FDR 유의 T1 1/9,
   비선형 HGB에선 copy가 예측을 **해침**(OOS marginal −0.023). → "copy 무의미"가 아니라 **"차트 안 위치는 문구로 설명 안 됨"**.
   설계상 copy가 가장 중요할 *발견(discovery) 단계*는 표본서 제외됨(한계).
2. **"베스트셀러/수상 키워드=판매"는 교란.** raw 2.0× 격차(ΔR² 0.051) → **장르만 통제해도 0.012로 급감** → full 통제 0.007,
   FDR 유의 0/8. badge↔리뷰 상관 0.04~0.16(낮음). → 순수 역인과라기보다 **장르 구성 교란**; 키워드는 잘 팔리는 장르와 동반될 뿐.
3. **장르 + 누적 리뷰가 지배**(이 표본 기준; 균형설계라 장르 우세 *크기*는 과대 가능). context(거의 장르) +0.215, 리뷰 +0.244.
   둘 다 copywriter 문구 lever 아님.
4. **사전런칭 예측 ≈ 장르평균.** 20시드: 최선의 출간전 모델도 장르평균 +0.007±0.009 (0/20이 +0.05 도달), **copy 넣으면 더 나빠짐**.
   "어떤 장르인가"가 거의 전부. 리뷰(출간후)가 장르 위 최대 추가 신호(B1−B2≈+0.10)지만 사전 lever 아님.
5. **검증할 가설 1개:** 책소개에 *구체적 숫자* 포함 → +12.5%/IQR (FDR q=0.02, Bonferroni·clustered SE 강건, 15/18 장르서 양).
   단 **관찰적·book-type 교란 가능·OOS 예측 도움 안 됨** → 입증된 lever 아니라 **A/B 테스트할 가설**.
   *(footnote: has_blurb b=−0.14는 collinearity artifact, FDR 미통과, 해석 안 함.)*

## 비즈니스 함의
차트 안 책 기준, **blurb 문구 미세조정에 과투자할 근거 없음** — 장르 포지셔닝 + 출간후 리뷰·입소문 모멘텀이 핵심.
"성취 배지를 박아라"식 copy 조언은 무효(있는 성취 기록은 결과지 원인 아님). 시도해볼 단 하나: 책소개에 *구체적 숫자/근거*(작은 효과, 가설).

## 한계 (결론을 뒤집을 수 있는 것)
- **결과변수-선택 표본**(차트인 + salesPoint>0): discovery margin 제외 → copy 효과가 *축소*됐을 수 있음. (Aladin 구조상 무편향 표집 불가)
- **텍스트=짧은 blurb(중앙값 142자)뿐** — 전체 책소개·목차·서평 미반영.
- **단일 스냅샷(2026-06-29)·단일 리테일러(알라딘)·SalesPoint≠매출**(최근가중 모멘텀 지수).
- **미관측 교란**(마케팅비·표지·미디어·각색) → copy 계수는 상한. 관찰적 → 상관, 인과 아님.
- 균형설계 표본 → genre 우세 *크기*는 인구비례 아님.
